# Proyecto PetShop

Este es el proyecto web **PetShop** para la entrega final del curso de CODERHOUSE, desarrollado con **HTML, CSS y JavaScript**.  
El proyecto consiste en 5 páginas web estáticas, para mostrar productos y funcionalidades básicas de un PetShop.
